Html-layout
Aloitetaan Html layout-koodilla. Tällä on helppo aloittaa html-sivu.
Tähän on myös käytössä komento "!".

<!DOCTYPE html>
<html lang="en">
<head >
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>

</body>
</html>

Css

Ensimmäiseksi linkataan css-tiedosto html:ään.

<head>

<link rel="stylesheet" type="text/css" href="tiedosto.css"/>

</head>

Tehdään body-elementille määritykset

body {
margin: 0;
padding: 0;
}

Tehdään buttonille määritykset

button {
background-color: gray
border: none;
color: white;
padding: 15px 32px;
text-align: center;
text-decoration: none;
display: inline-block;
font-size: 16px;
}

Javascript

Ensimmäiseksi linkataan js-tiedosto html:ään

<body>

<script src="tiedosto.js">

</body>

Alert
Tehdään alert-funktio kun buttonia painetaan.

Html
Lisätään buttoniin onclick() funktio.

<body>

<button onclick="alert()">Click Me>/button>

>/body>

Javascript

function alert() {
alert("Hello World!");
}
